Krypt is a Dungeon-focused Quality of Life module for Hypixel Skyblock.

## Features

### General

- Architect draft message
- Better party finder
- Blood room helper
- Breaker charge display
- Crypt reminder
- Custom mage beam
- Hide non-starred nametags
- Mask timers
- Phase tick timers
- Positional message
- Room name HUD
- Room secrets HUD
- Scarf spawn timers
- Secret sounds
- Server freeze indicator
- Split timers
- Spring boots overlay

### Alerts

- Bat death alert
- Fire freeze alert
- Key alerts
- Leap alert
- Melody alert
- Mimic alert
- Prince alert
- Score alert
- Shadow assassin alert
- Ultimate alerts

### Dungeon map

- a LOT of customisation available
- Dungeon breakdown at end of run

### Solvers

- Blaze
- Boulder
- Creeper beams
- Quiz
- Teleport Maze
- Three weirdos
- Tic-Tac-Toe
- Water Board
- Livid solver

### Floor 7

- Arrow-Align solver
- Necron health display
- Simons-Says solver
- Terminal tracker
- Terminal waypoints

### Highlights

- Door highlight
- Key highlight
- Star mob highlight
- Teammate highlight
- Wither highlight
